안녕하세요, 데이터 분석가로 일하고 있는 datastro입니다. 오랫동안 미뤄왔던 개인 블로그를 이제야 시작하게 되어, 첫 글로 이 공간을 어떤 용도로 채워나갈지 정리해보려 합니다.

## 다루는 주제

### 읽은 책

가장 자주 올라올 글은 독서 기록이 될 것 같습니다. 데이터 사이언스나 경제학 관련 전공 서적뿐 아니라, 실무와 직접 관련이 없는 책도 편하게 읽고 정리해보려 합니다. 완독 후의 총평보다는, 읽는 도중에 든 생각이나 인상 깊었던 구절을 짧게라도 남기는 방식으로 접근할 계획입니다.

### 데이터 분석 방법론에 대한 고민

두 번째로 중심이 될 주제는 **데이터 자체에 대한 고민**과 **분석 방법론에 대한 고민**입니다. 실무에서 분석을 하다 보면 특정 지표를 어떻게 정의할지, 어떤 모델링 방식이 지금 상황에 적합한지 매번 다시 고민하게 되는 순간들이 있습니다. 이런 고민의 과정을 정리해두면 나중에 비슷한 문제를 다시 마주쳤을 때 도움이 될 것 같습니다.

다만 회사에서 다루는 실제 데이터는 그대로 공유하기 어려운 경우가 많습니다. 그래서 공개된 샘플 데이터셋을 찾아 예시로 활용해보려고 합니다.

> 샘플 데이터를 구하면 아래와 같은 형태로 코드와 함께 정리해볼 계획입니다.

```python
import pandas as pd

df_sample = pd.read_csv("sample_data.csv")
df_sample.groupby("category")["value"].mean()
```

### 밤하늘 사진

가끔 별 사진을 찍으러 나가곤 하는데, 날씨와 시간이 맞아야 하는 일이라 자주 올리지는 못할 것 같습니다. 그래도 마음에 드는 사진이 나오면 촬영 장소나 설정값 같은 것도 함께 남겨보려 합니다.

### 기타

위 세 가지 주제 외에도, 커피처럼 소소하게 기록해두고 싶은 것들이 생기면 이 카테고리에 모아둘 예정입니다. 정해진 형식이나 주기는 따로 두지 않으려 합니다.

## 게시 계획 (콘텐츠별 예상 빈도)

블로그에 올라올 글의 종류를 대략 정리하면 아래와 같습니다.

| 카테고리 | 빈도 | 설명 |
| --- | --- | --- |
| 독서 | 가장 잦음 | 매일 조금씩이라도 읽고 있어서, 가장 꾸준히 올라올 주제입니다. |
| 데이터·분석 | 중간 | 업무나 개인 학습 중 들었던 고민을 정리한 글입니다. |
| 밤하늘 사진 | 드묾 | 촬영을 나갈 수 있는 여건이 맞아야 해서 자주는 어려울 것 같습니다. |
